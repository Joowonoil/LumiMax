## Why LumiMax?

LumiMax uses Apple's native EDR (Extended Dynamic Range) overlay to extend display brightness while preserving color accuracy.

|                    | Gamma adjustment         | LumiMax            |
| ------------------ | ------------------------ | ------------------ |
| **Method**         | Gamma table manipulation | Native EDR overlay |
| **Color accuracy** | Colors get distorted     | Preserved          |
| **Max brightness** | Limited                  | Full 1600 nits     |

## Features

### 🔆 XDR Brightness Boost

Push your MacBook Pro display up to 1600 nits — the true hardware maximum. Perfect for HDR content, bright outdoor environments, and sunlit offices.

### ⌨️ Keyboard Backlight Control

Adjust keyboard backlight brightness with a convenient slider.

### 🌙 Display Mode Controls

Quick toggles for Dark Mode, Night Shift, and True Tone right from the menu bar.

### 🖥️ Multi-Display Support

Control brightness for all connected displays including Pro Display XDR, LG UltraFine, Studio Display, and third-party monitors.

### 🎹 Keyboard Shortcuts

- **F1/F2** — Brightness control with keyboard
- **Shift+Option+F1/F2** — Fine-grained brightness adjustment

### 🔋 Smart Battery Protection

Auto-disable on battery or when charge drops below your set threshold. Timer-based auto-off also available.

### 🌍 7 Languages

English, Korean, Japanese, Chinese, German, Spanish, and French.

Want to help translate LumiMax into your language? [Open an issue](https://github.com/Joowonoil/LumiMax/issues) or email us at [ramterstudio@gmail.com](mailto:ramterstudio@gmail.com).

## Supported Devices

- MacBook Pro 14" / 16" (M1 Pro/Max and later) — Full XDR boost up to 1600 nits
- MacBook Pro 13" (M1/M2) — Optimized boost
- MacBook Air (M1 and later) — Up to +60% brightness boost
- Pro Display XDR, LG UltraFine, and other external displays
- Intel Macs — Universal binary with display-aware boost limits

## System Requirements

- **macOS 13.0+**
- Apple silicon or Intel Mac
- XDR display recommended for full 1600 nit boost
